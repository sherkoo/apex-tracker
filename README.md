# Apex Tracker App
A full stack Vue/Express utilizing Apex legends API to track player stats.

## Quick Start
```
$ npm install
```

```
$ cd client
$ npm install
```

## Development

Starting the server:

```
$ npm run dev
```
This will run the back-end and the front-end simultaneously.

## Build
```
$ npm run build
```
This will build in ```/public```
